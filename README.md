# CIFAR-10 Classification

This repository contains code for training a convolutional neural network on the CIFAR-10 dataset.

## Dataset
CIFAR-10 is a standard benchmark dataset consisting of 60,000 32x32 color images in 10 classes.  
50,000 images are used for training and 10,000 for testing.  

The dataset is downloaded automatically by the code using `torchvision.datasets.CIFAR10`.  

Official CIFAR-10 page: [https://www.cs.toronto.edu/~kriz/cifar.html](https://www.cs.toronto.edu/~kriz/cifar.html)
